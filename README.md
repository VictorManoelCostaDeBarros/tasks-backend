# tasks-backend
this is the backend of application ReactNative 
with nodejs, knex and postgresql

## Instalação

Você precisará ter apenas o [NodeJS](https://nodejs.org) instalado na sua máquina, e após isso, clonar este repositório:
```bash
  $ git clone https://github.com/VictorManoelCostaDeBarros/tasks-backend.git
```

Depois acesse o diretório e instale as dependências executando o seguinte comando:
```bash
  $ yarn install # ou npm install
```

## Executando a aplicação

Execute o comando a baixo para inicializar o Next:
```bash
  $ yarn start
```
